> html5, scss and js gulp boilerplate for fast prototyping

### Installation

Open your terminal and type in

```sh
$ git clone https://github.com/Yilber/html5-css-js-boilarplate.git
$ cd html5-css-js-boilarplate
```

Install all the packages

```sh
$ npm install
```
